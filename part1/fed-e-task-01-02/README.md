### 4. 请详细说明var、let、const三种声明变量的方式之间的具体差别
|      |  var   | let  | const |
| ---- |  ----  | ----  | ---- |
| 语法使用 | var a = 1  | let a = 1 | const a = 1 |
| 声明提前 | 声明提前使用undefined  | 声明提前，不初始化 | 声明提前，不初始化 |
| 作用域 | 全局  | 块级作用域 | 块级作用域 |
| 重复定义 | 可以  | 不可以 | 不可以 |
| 多次赋值 | 可以  | 可以 | 不可以 |
| 声明前访问 | 可以  | 不可以 | 不可以 |
